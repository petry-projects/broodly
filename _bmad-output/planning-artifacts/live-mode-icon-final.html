Design Rationale

RecognizabilityEQ bars are universally understood as "audio/live" across all platforms — Spotify, podcasts, voice assistants. Zero learning curve.
No bee neededThe app logo (bee) already sits in the header. Doubling it would add clutter without information. The EQ bars add a distinct, complementary meaning.
Brand alignmentThe amber gradient is unique to Broodly. No other app uses this exact warm-to-deep amber progression for audio — it becomes an ownable visual signature.
Scales cleanly5 simple rects with rounded ends remain crisp at every size from 20px to 48px. No detail is lost at the 26px app bar size.
AnimatableActive state uses CSS scaleY on each bar with staggered delays. Lightweight, respects prefers-reduced-motion (falls back to static).
AccessibleStrong contrast on both light and dark backgrounds. Clear state differentiation (amber = active, grey = disabled). Paired with aria-label.
